# Audio Files – Nông Trại Xanh v5

Đặt các file MP3 sau vào thư mục này:

| File              | Mô tả                                       | Kích hoạt khi                     |
|-------------------|---------------------------------------------|-----------------------------------|
| water.mp3         | Tiếng nước tưới (watering can pour)         | Tưới nước cây                     |
| sleep.mp3         | Tiếng tắt công tắc / gió nhẹ (wind-down)   | Nhấn nút ngủ 🌙                   |
| rooster.mp3       | Tiếng gà gáy (cock-a-doodle-doo)            | Sáng thức dậy mỗi ngày mới        |
| fishing1.mp3      | Tiếng nước câu cá – phiên bản 1             | Câu cá (ngẫu nhiên 1/3)           |
| fishing2.mp3      | Tiếng nước câu cá – phiên bản 2             | Câu cá (ngẫu nhiên 2/3)           |
| fishing3.mp3      | Tiếng nước câu cá – phiên bản 3             | Câu cá (ngẫu nhiên 3/3)           |
| coins.mp3         | Tiếng xu vàng (coin jingle)                 | Thu hoạch cây / bán đồ            |
| hoe.mp3           | Tiếng cuốc đất + gieo hạt (dig/hoe)        | Cuốc đất & gieo hạt               |
| feed.mp3          | Tiếng cho ăn (animal feed / munch)          | Cho gà/bò ăn                      |
| day_bgm.mp3       | Nhạc nền ban ngày (calm daytime BGM, loop)  | Suốt thời gian chơi ban ngày      |
| night_bgm.mp3     | Nhạc nền ban đêm (peaceful night, loop)     | Trong khoảng chuyển đêm → ngày    |

## Ghi chú
- Tất cả file phải là định dạng MP3.
- day_bgm.mp3 và night_bgm.mp3 sẽ được loop tự động.
- Nếu file bị thiếu, game vẫn chạy bình thường (chỉ không có âm thanh đó).
