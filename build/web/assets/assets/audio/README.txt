# Audio Assets for Farm Game v5

Place these MP3 files in this folder (assets/audio/):

| File            | Description                        | Suggested source               |
|-----------------|------------------------------------|--------------------------------|
| water.mp3    .  | Watering-can pouring sound         | freesound.org – search "water pour" |
| sleep.mp3     . | Light-switch click / wind-down     | freesound.org – search "click switch" |
| rooster.mp3    .| Morning cock-a-doodle-doo          | freesound.org – search "rooster crow" |
| fishing1.mp3  . | Fishing splash 1                   | freesound.org – search "water splash" |
| fishing2.mp3   .| Fishing splash 2                   | freesound.org – search "splash" |
| fishing3.mp3 .  | Fishing splash 3                   | freesound.org – search "fish caught" |
| coins.mp3     . | Coin jingle (harvest/sell)         | freesound.org – search "coins jingle" |
| hoe.mp3         | Digging / hoe sound                | freesound.org – search "digging dirt" |
| feed.mp3   .    | Animal feed / munch                | freesound.org – search "crunch eating" |
| day_bgm.mp3     | Calm daytime background music      | freesound.org – search "farm ambient" |
| night_bgm.mp3   | Peaceful night music               | freesound.org – search "night ambient" |

All audio files are OPTIONAL.  The game works silently if files are missing.
Free, royalty-free MP3s can be downloaded from:
  - https://freesound.org  (CC0 / CC-BY license)
  - https://opengameart.org
  - https://pixabay.com/music/
